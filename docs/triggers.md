# 动作触发器

### onInteract

玩家交互时触发。

**别名：** `interact` 、 `Interact`

**内置变量：**

| 变量名   | 描述         |
| -------- | ------------ |
| `event`  | 事件         |
| `player` | 谁交互了     |
| `item`   | Ratziel 物品 |


#### onLeft

左键交互触发。

**别名：** `left`

#### onRight

**别名：** `right`
