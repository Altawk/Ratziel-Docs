# 物品标签

:::info
使用**物品标签**是快捷调用物品数据，生成物品信息的一种方法。

分为两种：**片段 (Section)** 和 **标签 (Tag)**，统称 **物品标签** (真想不到啥好的中文名了)
:::

## 标签 (Tag)

**一般用法：** `{解析器名称:参数1:参数2:...}`

**详见：** [标签解析器](../quick-lookup/tag-resolver.md)

## 片段 (Section)

**详见：** [片段解析器](../quick-lookup/section-resolver.md)
