# 配置插件

若插件成功安装，你将会在服务端插件目录下看到一个名为 `Ratziel` 的目录，其结构如下。

```text
Ratziel
├── lang ···················· 语言文件目录
├── workspace ··············· 工作空间目录
├── settings.yml ············ 插件配置文件
└── kether.yml ·············· Kether 配置文件
```
